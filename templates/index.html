How to Use

  1. Select Project: Choose from BMS, Battery TCU, or Vehicle TCU
  2. Upload Inventory: Provide Excel file with MPN and Qty columns
  3. Set Parameters: Enter build quantity and wastage percentage
  4. Calculate: Click to compute shortages immediately
  5. Export: Download results as Excel file for documentation

💡 Note: The system automatically handles alternate MPNs by aggregating quantities from all matching part numbers.
